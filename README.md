# hexo-blog
blog build with hexo

## step

### git clone
```
git clone git@github.com:gongph/hexo-blog.git
```


### install

global install hexo-cli
```
npm install hexo-cli -g
```

local install modules
```
npm install 
```

### deploy
```
hexo deploy
```

### start
```
hexo server // http://localhost:4000/
```
